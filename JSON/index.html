JSON test for remembering Levels

Steps:

1 - Initialise node.js and install express in this directory (npm init, npm install express --save)

2 - type "node index.js" in CMD

3 - Go to "http://localhost:8082/" to see Current Level and Highest Completed Level.

4 - Refresh the page to observe the Current Level Value increasing.

5 - Highest Level Value will increase if Current Level surpasses it.

6 - To reset current Level, go to "http://localhost:8082/reset".

7 - Highest Level will keep the value of the latest level completed.
