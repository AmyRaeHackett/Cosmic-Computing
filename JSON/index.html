JSON test for App. Need to use console for test. Will not work in browser.

Steps:

1 - Install node js in this directory (npm init)

2 - type "node index.js" in CMD

3 - CMD will display the JSON entry { level: 4 }

4 - Change the value of currentLevel in the index.js file (this represents the current level being played.

5 - if this value is lower than 4, the JSON value will not update (as level 1, 2, & 3 are complete)

6 - if this value is greater than 4, the JSON value will update to that value (as the highest level completed now surpasses 4)

7 - running "node index.js" again should update to reflect this higher value.
